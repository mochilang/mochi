Use binary search with a duplicate-safe fallback. Compare the middle element with the right edge. If `nums[mid] < nums[right]`, the minimum is in the left half including `mid`. If `nums[mid] > nums[right]`, the minimum is in the right half after `mid`. When `nums[mid] == nums[right]`, duplicates hide the sorted side, so shrink `right` by one and continue. The interval eventually collapses to the minimum element.
