Use the greedy backtracking matcher. Walk through `s` and `p` with indices `i` and `j`. When characters match directly, or the pattern has `?`, advance both. When the pattern has `*`, remember its position in `star` and remember the current string position in `match`, then advance the pattern. If a later mismatch happens and we have seen a `*`, backtrack only to the most recent star: let that star absorb one more character by incrementing `match`, reset `i = match`, and continue from `j = star + 1`. After the string is consumed, any remaining pattern characters must all be `*`. This runs in `O(n + m)` average practice and uses `O(1)` extra space.
