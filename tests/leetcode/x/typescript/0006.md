# Zigzag Conversion

Use the row-by-row cycle pattern.

If `numRows` is `1` or at least the string length, the answer is the original string. Otherwise the zigzag repeats every `cycle = 2 * numRows - 2` characters.

For each row, take the vertical character at positions `row, row + cycle, row + 2 * cycle, ...`. For middle rows, also take the diagonal character inside the same cycle at `i + cycle - 2 * row` when it exists.

Appending rows in order produces the converted string in `O(n)` time.
