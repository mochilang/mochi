Store the tree in level-order with explicit `null` markers, then run a DFS by array index. For each node, compute the best downward path sum that can be extended to its parent: `value + max(0, leftGain, rightGain)`. At the same time, update a global answer with the best path that passes through the node: `value + max(0, leftGain) + max(0, rightGain)`. The global maximum over all nodes is the answer.
