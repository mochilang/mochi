# 0001 Two Sum

For each test case, scan every pair of positions `(i, j)` with `i < j`.
The first pair whose values sum to `target` is the required answer because the
problem guarantees exactly one valid pair.

The checked-in harness uses a simple format:

- first line: number of test cases
- per case: one line with `n target`
- next line: `n` integers

Each program prints the two answer indices on one line.
