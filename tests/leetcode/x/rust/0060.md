Use the factorial number system. Build the digit list `1..n`, convert `k` to zero-based by subtracting `1`, and at each position compute `block = (remaining-1)!`. The quotient `k / block` tells which unused digit to take next, then remove that digit and reduce `k %= block`. Repeating this from left to right constructs the exact `k`-th lexicographic permutation without generating all permutations.
