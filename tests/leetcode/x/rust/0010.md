# Regular Expression Matching

Use recursion on positions in the input string and pattern.

At each pattern position, first check whether the current characters match. A normal character must be equal, and `.` matches any single character. The only special branch is when the next pattern character is `*`.

If there is a `*`, there are two choices: skip the whole `x*` block, or consume one matching character from the string and stay on the same pattern position so `*` can match more characters. If there is no `*`, both the string and pattern just advance by one.
