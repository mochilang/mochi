Use the standard two-pass greedy assignment. Give every child `1` candy first. Sweep left to right and if `ratings[i] > ratings[i-1]`, set `candies[i] = candies[i-1] + 1`. Then sweep right to left and if `ratings[i] > ratings[i+1]`, raise `candies[i]` to at least `candies[i+1] + 1`. Summing the final array gives the minimum valid distribution because each pass enforces one neighbor constraint and the `max` keeps both constraints satisfied.
