Use dynamic programming on prefixes. Precompute which substrings `s[i..j]` are palindromes, usually by expanding from shorter substrings to longer ones. Then let `cuts[i]` be the minimum cuts needed for `s[0..i]`. If `s[0..i]` itself is a palindrome, `cuts[i] = 0`; otherwise try every last cut position `j < i` where `s[j+1..i]` is a palindrome and minimize `cuts[j] + 1`.
