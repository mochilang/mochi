Use backtracking row by row with three occupancy structures: `cols[c]` for used columns, `diag1[r+c]` for the main diagonal, and `diag2[r-c+n-1]` for the anti-diagonal. At each row, try every column that is free in all three structures, recurse, and add up the counts returned by deeper calls. This version does not need to store whole boards; it only returns the number of valid completions for each test case.
