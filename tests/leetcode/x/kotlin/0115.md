Let `dp[j]` be the number of ways to form the first `j` characters of `t` using the processed prefix of `s`. Start with `dp[0] = 1` because the empty target can always be formed once. For each character of `s`, walk `j` from right to left. If `s[i] == t[j-1]`, then every way that previously formed `t[0..j-1)` can extend to a way that forms `t[0..j)`, so add `dp[j-1]` into `dp[j]`. The final answer is `dp[len(t)]`.
