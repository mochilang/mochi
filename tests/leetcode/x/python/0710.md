Let `bound = n - len(blacklist)`. We want every pick to land uniformly in `[0, bound)`, then translate any blacklisted value from that range to some allowed value in `[bound, n)`. So build a map only for blacklisted numbers below `bound`, pairing each with an unused allowed tail value. Then `pick()` draws an index in `[0, bound)` and returns either that index or its remapped partner. For this harness, the random draw is made deterministic with a fixed 32-bit LCG seed so all languages emit the same sequence.
