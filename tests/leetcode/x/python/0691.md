Treat the target as a bitmask of uncovered positions. Precompute letter counts for every sticker, then run DFS with memoization over masks: from each mask, try every sticker that can cover the first still-missing target character, greedily advance the mask as far as that sticker can, and recurse. Because the target is short, there are only `2^m` states, so memoizing the minimum sticker count for each mask makes the search tractable.
