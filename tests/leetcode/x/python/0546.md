Use memoized dynamic programming on intervals. Let `dp(l, r, k)` be the maximum score obtainable from `boxes[l..r]` if there are already `k` extra boxes equal to `boxes[r]` attached to the right. First compress any equal suffix at the right end into that `k` count. One option is to remove that whole suffix immediately, giving `dp(l, r-1, 0) + (k+1)^2`. The better option sometimes is to keep `boxes[r]` and merge it with an earlier matching color `boxes[i]`, clearing the middle first and then solving `dp(l, i, k+1)`. The memo table makes this exponential-looking search fast enough.
