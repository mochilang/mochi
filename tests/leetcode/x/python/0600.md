Let `f[i]` be the number of valid binary strings of length `i` with no consecutive ones; this follows the Fibonacci recurrence because a valid string either starts with `0` and then any valid length-`i-1` suffix, or starts with `10` and then any valid length-`i-2` suffix. Then scan the bits of `n` from high to low. Whenever a bit is `1`, add `f[i]` for the count of valid numbers formed by putting `0` at that position and anything valid below it. If two consecutive `1` bits appear in `n`, stop immediately because every larger suffix choice is invalid. If the scan finishes, include `n` itself.
