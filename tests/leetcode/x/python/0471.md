Let `dp[i][j]` be the shortest encoding for the substring `s[i:j+1]`. For each interval, start with the raw substring, then try every split `k` to combine `dp[i][k] + dp[k+1][j]`, and finally check whether the whole interval is several copies of a smaller pattern so it can become `count[dp(pattern)]`. Ties are broken lexicographically so the fixture output stays deterministic across languages.
