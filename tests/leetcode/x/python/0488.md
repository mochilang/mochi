Use depth-first search over the current board and remaining hand counts, with memoization on both pieces of state. At each maximal run of one color, try spending just enough balls to make that run reach length three, then remove it and repeatedly collapse any new groups created by the chain reaction. Because the search only ever inserts the minimum useful number of balls into a run, the branching factor stays manageable and memoization turns repeated subproblems into cache hits.
