Use dynamic programming on prefixes. Let `dp[i]` be the number of ways to decode the first `i` characters. Each step combines the ways to decode the last character alone and the last two characters together. The only real work is counting how many decodings a single symbol or a pair can contribute when `'*'` may stand for digits `1` through `9`. Because each state depends only on the previous two states, constant space is enough.
