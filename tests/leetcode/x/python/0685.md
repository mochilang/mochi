There are only three structural cases: a pure cycle, a pure two-parent conflict, or both at once. First scan the edges to see whether some node gets two different parents, recording the earlier and later conflicting edges. Then run union-find over the directed edges while optionally skipping the later conflicting edge. If a cycle still appears, the earlier conflicting edge is the answer; if no cycle appears, the later conflicting edge is the answer; and if there was no two-parent conflict at all, the edge that closes the cycle is the answer.
