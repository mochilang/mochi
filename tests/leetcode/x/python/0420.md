Count missing character classes and runs of three or more equal characters. Short passwords need insertions, so the answer is max(missing_classes, 6 - length). For length 6..20, replacements break repeated runs, so use max(missing_classes, sum(run // 3)). For overlong passwords, deletions are mandatory; spend them first on runs where one or two deletions reduce a replacement, then use remaining deletions in groups of three, and finally add the larger of missing classes or remaining replacements.
