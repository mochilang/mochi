Precompute the ring indices for each character. Then run dynamic programming over the key: `dp[pos]` is the minimum rotation cost to finish the prefix of the key while leaving ring index `pos` at twelve o'clock. To advance one character, try every matching destination index for that character, add the shortest circular distance from each current position, and keep the best value; after all key characters are processed, add `len(key)` for the button presses.
