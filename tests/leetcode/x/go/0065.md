Scan the string once while tracking four facts: whether a digit has appeared at all, whether a dot has appeared, whether an exponent marker has appeared, and whether a digit has appeared after the exponent. A sign is allowed only at the start or immediately after `e`/`E`, a dot is allowed only before any exponent and only once, and `e`/`E` is allowed only once and only after some digit. At the end, the string is valid exactly when it has digits overall and, if an exponent was used, digits after the exponent too.
