# Reverse Integer

Build the reversed number one digit at a time.

Repeatedly pop the last digit with `% 10` and push it onto the answer with `ans = ans * 10 + digit`. Before pushing, check whether multiplying by `10` and adding the next digit would move outside the signed 32-bit range `[-2147483648, 2147483647]`.

If an overflow would happen, return `0`. Otherwise continue until the input becomes `0`.
