# Generated Fortran Programs

This directory contains Fortran source code and outputs generated by the compiler tests. Each Mochi program from `tests/vm/valid` is compiled to a Fortran `.f90` file. Successful runs have a corresponding `.out` file. If compilation or execution fails, an `.error` file is written.
