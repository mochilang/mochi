# Generated Pascal Programs

This directory contains Pascal source code and outputs generated by the compiler tests. Each Mochi program from `tests/vm/valid` is compiled to a `.pas` file. Successful runs have a corresponding `.out` file. If compilation or execution fails, an `.error` file is written.
