# SPOJ FACT1 - Factorial

https://www.spoj.com/problems/FACT1/

Given n, compute the sum of digits of n!.

## Approach

Simulate big integer multiplication (as a list of decimal digits).
Multiply 1 * 2 * 3 * ... * n digit by digit with carry.
Sum all digits at the end.

## Edge cases

- n = 0 or 1: 1! = 0! = 1, digit sum = 1.
- n up to 10000 (problem says up to some limit; handle large factorials).
