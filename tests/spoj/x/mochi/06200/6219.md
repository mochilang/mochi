# SPOJ Edit Distance (EDIST)

https://www.spoj.com/problems/EDIST/

Compute the minimum edit distance (Levenshtein) between two strings.

## Approach

Standard O(n*m) DP: dp[i][j] = edit distance between s[0:i] and t[0:j].
dp[i][j] = dp[i-1][j-1] if s[i]==t[j], else 1 + min(dp[i-1][j-1], dp[i-1][j], dp[i][j-1]).
