# SPOJ FAVDICE - Favorite Dice

https://www.spoj.com/problems/FAVDICE/

Expected number of rolls to see all n faces of a fair die (coupon collector's problem).
E(n) = n * (1 + 1/2 + 1/3 + ... + 1/n) = n * H(n).

## Approach

Compute the n-th harmonic number H(n) as a float, multiply by n, then format
to 2 decimal places using the fmtDecimal helper (multiply by 100, add 0.5,
truncate to int, extract integer and fractional parts).
