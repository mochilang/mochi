# SPOJ Bitmap (BITMAP)

https://www.spoj.com/problems/BITMAP/

Given a binary grid of 0s and 1s, find for each cell the distance (Manhattan/BFS steps
in 4 directions) to the nearest cell with value 1.

## Approach

Multi-source BFS: initialize all cells with value 1 at distance 0, all other cells at
infinity. Use a queue (list with head pointer). Process each cell by expanding to its
4 neighbors. If the neighbor has not yet been reached, set its distance and enqueue it.
Output the distance grid, space-separated per row.
