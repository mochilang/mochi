# SPOJ Bytelandian Gold Coins (COINS)

https://www.spoj.com/problems/COINS/

A coin worth n can be exchanged for coins worth n/2 + n/3 + n/4.
Find the maximum total value obtainable from one coin worth n.

## Approach

Memoized recursion: solve(n) = max(n, solve(n/2) + solve(n/3) + solve(n/4)).
Use a map for memoization since n can be up to 10^9.
