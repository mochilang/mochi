# SPOJ PERMUT2 - Ambiguous Permutations

https://www.spoj.com/problems/PERMUT2/

A permutation P (1-indexed) is "ambiguous" if P[P[i]] = i for all i,
meaning the permutation equals its own inverse.

## Approach

Read each permutation into a list, then for every index i check that
P[P[i]-1]-1 == i (converting to 0-indexed). If all positions satisfy this,
output "ambiguous", else "not ambiguous".
