# SPOJ CANTON - Count on Cantor

https://www.spoj.com/problems/CANTON/

Cantor's diagonal enumeration of rational pairs (a, b):
diagonal d = a + b - 1, with offset = (d-1)*d/2 elements before it.
If d is odd, the diagonal goes top-right so position = offset + b.
If d is even, the diagonal goes bottom-left so position = offset + a.

## Approach

Parse "a/b" from each line, compute diagonal d = a+b-1, offset = d*(d-1)/2,
then use parity of d to get final position.
