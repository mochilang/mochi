# SPOJ BFBIGTHN - "If Bigger Than" in BF

https://www.spoj.com/problems/BFBIGTHN/

Read a two-digit count t, then t lines each with two-digit numbers a and b. Output 1 if a > b, else 0, one result per line.

## Approach

Parse t from first line, then loop t times reading "aa bb" pairs. Compare integers directly and print 1 or 0.
