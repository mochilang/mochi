# SPOJ HS09BF - Primes in Brainfuck (Mochi implementation)

https://www.spoj.com/problems/HS09BF/

No input. Print all prime numbers up to 100 on a single line, separated by commas. Score is source code length.

## Approach

Sieve of Eratosthenes up to 100. Collect all primes and join them with commas, then print once.
