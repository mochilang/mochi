# Strchr in BF

**Slug:** `BFSTR1` | **Section:** challenge | **Rank:** 19920
**URL:** https://www.spoj.com/problems/BFSTR1/

Given T test cases. Each line: character C, space, string STR.
Find first 1-indexed occurrence of C in STR. Print 0 if not found.
Score is source code length (only 256 bytes allowed).
