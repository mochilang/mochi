# SPOJ AOC2512 - AoC Christmas Tree Farm Part 3

https://www.spoj.com/problems/AOC2512/

Pack K shapes (3x3 grids) onto an unlimited grid, maximizing covered_area / bounding_area.

## Approach

Naive placement: each shape instance is placed at (x=3*i, y=0) with identity transformation (ID 0). This puts all shapes in a horizontal row, with no overlap since shapes are 3 wide.

Input parsing:
- Shape definition lines contain "#", ".", or "k:" format (no "x" character).
- Challenge lines have format "NxM: a_0 a_1 ..." and contain "x".
- Read until 3 consecutive empty lines (to handle EOF), collecting challenge lines.
- For each challenge: extract counts a_i, emit "shapeID 0 x 0" placements with x += 3.

## Notes

Score = 100 * covered_area / area. Naive horizontal placement gives score proportional
to the density of '#' cells vs total 3-wide strip. For solid shapes (all '#'), score = 100%.
Output order within a challenge does not matter. Challenges are separated by blank lines.
