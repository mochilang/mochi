# SPOJ SBSTR1 - Substring Check (Bug Funny)

https://www.spoj.com/problems/SBSTR1/

Given two binary strings A and B on a line, print 1 if B is a substring of A, else 0.

## Approach

Simple sliding window: check every position in A for a match with B.

## Notes

Input has no T count. Read until EOF, each line has two space-separated strings.
