# SPOJ Hash it! (HASHIT)

https://www.spoj.com/problems/HASHIT/

Open addressing hash table with 101 slots. Hash: h(key)=19*sum(ASCII(ai)*i) mod 101.
Collision resolution: (h + j^2 + 23*j) mod 101, j=1..19 (at most 20 probes).
Operations: ADD:key (ignore duplicate), DEL:key (mark slot empty, ignore missing).

## Approach

Build ASCII lookup map for A-Z, a-z. For each test case, initialize 101 slots to "".
Use tc-prefixed map keys (str(tc)+":"+str(idx)) to avoid VM map persistence bug.
For ADD: probe sequence, insert at first empty slot. For DEL: probe until found or empty, clear.
Output: count, then index:key pairs sorted ascending by index.
