# SPOJ Hash it! (HASHIT)

https://www.spoj.com/problems/HASHIT/

Simulate a hash table with open addressing (linear probing). Operations: INSERT key,
DELETE key, SEARCH key. The hash function is sum of ASCII values mod 101.

## Approach

Maintain an array of 101 slots. Each slot is either empty, deleted, or has a key.
- INSERT: hash(key), probe linearly, place in first empty or deleted slot.
- DELETE: hash(key), probe linearly until key found or empty slot; mark as deleted.
- SEARCH: hash(key), probe linearly until key found or empty slot; report found/not.
