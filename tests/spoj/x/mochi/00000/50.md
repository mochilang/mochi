# SPOJ Invitation Cards (INCARDS)

https://www.spoj.com/problems/INCARDS/

Directed graph. For each test case, compute the minimum total cost for all P party members to travel from stop 1 to their assigned stop, then return to stop 1.

## Approach

Run Dijkstra twice:
1. Forward graph from stop 1: gives dist_fwd[v] = cost(1 -> v) for all v.
2. Reversed graph from stop 1: gives dist_rev[v] = cost(v -> 1) for all v.

Total = sum of (dist_fwd[v] + dist_rev[v]) for all v (vertex 1 contributes 0).

Use O(n^2) Dijkstra with linear min-extraction (suitable for dense graphs).
