# SPOJ Invitation Cards (INVIT1)

https://www.spoj.com/problems/INVIT1/

Directed graph. Find the minimum total cost for all party members to travel from stop 1
to their destination stop, plus the cost for each to return to stop 1.

## Approach

Run Dijkstra twice:
1. Forward graph from stop 1: gives cost(1 -> i) for all i.
2. Reversed graph from stop 1: gives cost(i -> 1) = cost on reversed graph from 1.

Total = sum of (forward[i] + reverse[i]) for i = 2..n.

Since edges have weights, use Dijkstra with a simple linear min-extraction (O(n^2)).
