# SPOJ POUR1 - Pouring water

https://www.spoj.com/problems/POUR1/

Three jugs with capacities a, b, c (c always full). Find minimum pours to measure
exactly d liters. If impossible, output -1.

## Approach

BFS on states (va, vb, vc) where vi is amount in jug i.
Initial state: (0, 0, c). Goal: any state where va=d or vb=d or vc=d.
Transitions: pour from any jug to any other (fill target or empty source).
