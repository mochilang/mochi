# SPOJ LABYR1 - Labyrinth

https://www.spoj.com/problems/LABYR1/

Find the diameter of a maze (longest shortest path between any two passable cells).
The maze is a grid of '#' (wall) and ' ' (open). Movement is 4-directional.

## Approach

Double BFS: 
1. BFS from any passable cell to find the farthest cell A.
2. BFS from A to find the farthest cell B and the distance = diameter.

## Notes

Multiple test cases. Input rows have varying lengths (pad with spaces to max width).
