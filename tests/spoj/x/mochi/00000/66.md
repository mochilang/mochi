# SPOJ Cross-country (CRSCNTRY)

https://www.spoj.com/problems/CRSCNTRY/

Given N skiers each with 3 leg times, form the fastest relay team by picking
the best skier for each leg independently (minimizing total time).

## Approach

For each of the 3 legs, find the minimum time across all skiers.
Output the sum of the three minimums. Multiple test cases until N=0.
