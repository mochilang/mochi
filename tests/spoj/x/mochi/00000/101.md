# SPOJ Fishmonger (FISHER)

https://www.spoj.com/problems/FISHER/

Find minimum travel time from city 1 to city N with total toll <= T.
Input: N cities, budget T, then N×N time matrix and N×N toll matrix.

## Approach

DP: dp[city][toll_spent] = minimum time to reach city using exactly toll_spent.
Initialize dp[0][0]=0, all others INF. Relax all edges Bellman-Ford style
(N-1 passes) to handle arbitrary directed graphs. Answer is min dp[N-1][s] for s<=T.
