# SPOJ Shopping (SHOP)

https://www.spoj.com/problems/SHOP/

Given a city grid map of width W and height H, find the minimum-cost path from
start cell S to destination cell D. Cell values are digits (0-9) representing the
cost to enter that cell. Cells marked X are impassable. S and D have zero entry cost.
Input: multiple test cases, each starting with W H, then H rows of the grid,
terminated by "0 0".

## Approach

Dijkstra's algorithm on the W×H grid. From each cell, try moving to 4 neighbors
(up, down, left, right). The edge cost is the digit value of the destination cell
(0 for S and D, blocked for X). Use O(n^2) Dijkstra (linear min-extraction).
