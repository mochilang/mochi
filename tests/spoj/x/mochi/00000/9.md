# SPOJ DIRVS - Direct Visibility

https://www.spoj.com/problems/DIRVS/

Given a 2D height grid, find the shortest path between two cells.
A cell is reachable from another if either cell is directly visible from start or end,
and the height difference going from one to a neighbor is at most +1 or -3.

## Approach

BFS on the grid. A cell is visitable if it is directly visible from either endpoint
(line-of-sight check using 3D ray marching above the height field).

## Notes

Expensive visibility check (O(P*Q) per cell). Used step-size sampling for ray check.
