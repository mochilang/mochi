# SPOJ CMPLS - Complete the Sequence!

https://www.spoj.com/problems/CMPLS/

Given S terms of a polynomial sequence and C more to predict, output the next C terms.

## Approach

Build the difference table (iterated finite differences). The bottom level is constant.
Extend each level by one term using the level below, working back up to the top.

## Notes

Polynomial sequences of any degree up to S-1. O(S*C) time with the difference table.
