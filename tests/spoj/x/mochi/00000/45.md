# SPOJ Commedia dell'Arte (COMMEDIA)

https://www.spoj.com/problems/COMMEDIA/

Count the number of ways to assign N actors to N distinct roles in a perfect matching,
given which roles each actor can play.

## Approach

Bitmask DP counting perfect matchings in a bipartite graph. Let dp[mask] = number of
ways to assign roles (those in mask) to the first popcount(mask) actors. Transition:
for each role r in mask that actor[popcount(mask)-1] can play, add dp[mask ^ (1<<r)].
Since Mochi lacks bitwise operators, implement bitGet and pow2 via arithmetic.

Input: T test cases. Each: N on a line, then N lines each starting with k (number of
roles this actor can play) followed by k role numbers (1-indexed).
Output: count of perfect matchings.

## Sample

- 2 actors each can play both of 2 roles -> 2 (swap assignments)
- 3 actors with partial role overlap -> 2 perfect matchings
- 2 actors where actor 1 plays role 1, actor 2 plays role 2 -> 1
