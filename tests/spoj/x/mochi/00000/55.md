# SPOJ Jasiek (JASIEK)

https://www.spoj.com/problems/JASIEK/

Count the number of ways to tile an m x n rectangular grid with 1x2 dominoes.

## Approach

Profile (bitmask) DP processed column by column. The state is a bitmask of which
rows in the current column are already pre-filled by horizontal dominoes extending
from the previous column. For each column, recursively try placing horizontal
dominoes (extending into the next column) or vertical dominoes (filling two adjacent
rows in the current column) in all unfilled rows. The answer is dp[0] after processing
all columns, starting with dp[0]=1. If m*n is odd, the answer is 0. Since Mochi lacks
bitwise operators, bit checks use division and modulo with a precomputed pow2 table.
