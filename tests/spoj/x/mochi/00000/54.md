# SPOJ JULKA - Julka

https://www.spoj.com/problems/JULKA/

Two children share n chocolates. Klaudia gets k more than Natalia.
Given total n and difference k, find each child's share.
Klaudia = (n+k)/2, Natalia = (n-k)/2.

## Approach

Big integer arithmetic: add/subtract the two numbers, divide by 2.
Numbers can be up to 10^100 digits.

## Edge cases

- Both n and k are large (up to 10^100). Use string big-int arithmetic.
- n+k is always even (both same parity since they're both positive integers of the same parity).
