# SPOJ Fake Tournament (TOUR)

https://www.spoj.com/problems/TOUR/

Given match results in a tournament, determine if there is a consistent ranking
(a permutation of teams such that every winner beats every lower-ranked team in the results).

## Approach

Model as a directed graph where edge u->v means u beats v. A consistent ranking exists
if and only if the graph has a unique topological order (no ties, no cycles). This happens
when the graph's topological sort is a Hamiltonian path - every consecutive pair is connected.

Algorithm:
1. Kahn's algorithm for topological sort (BFS with in-degree counts).
2. At each step, there must be exactly 1 node with in-degree 0.
   If ever 0 or 2+ nodes have in-degree 0, the ranking is not unique (output "No").
3. If we process all n nodes with exactly 1 choice at each step: output the order.
