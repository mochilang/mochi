# SPOJ PRIME1 - Prime Generator

https://www.spoj.com/problems/PRIME1/

For T test cases, given m and n, print all primes in [m, n], one per line, blank line between cases.

## Approach

Sieve of Eratosthenes up to sqrt(10^9) = 31623 to get small primes.
Then segmented sieve on [m, n] using those small primes.

## Notes

n can be up to 10^9, so a full sieve is impractical. Segmented sieve works in O((n-m) * log log n).
