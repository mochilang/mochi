# SPOJ The Shortest Path (SHPATH)

https://www.spoj.com/problems/SHPATH/

Weighted directed graph with city names. For each query (S, T), find the shortest path
cost from S to T.

## Approach

Dijkstra's algorithm with O(n^2) min-scan. Build CSR adjacency lists from city names
(using a map for name-to-index). For each query, run Dijkstra from the source.
