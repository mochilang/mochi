# SPOJ ARITH - Simple Arithmetics

https://www.spoj.com/problems/ARITH/

Given an arithmetic expression a op b (op = +, -, *), print the long-hand columnar
calculation as it would appear on paper.

## Approach

- For +/-: align numbers right, print separator, print result.
- For *: print each partial product (one per digit of b), a separator, and the total.

Widths are computed to right-align all rows.

## Notes

Big integer string arithmetic for addition/subtraction/multiplication.
The hardest part is formatting the columns correctly.
