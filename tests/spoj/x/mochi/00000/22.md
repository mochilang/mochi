# SPOJ TRGCNT - Triangle From Centroid

https://www.spoj.com/problems/TRGCNT/

Given the distances from the centroid to each vertex (m1, m2, m3, which are the
medians), find the area of the triangle.

The centroid divides each median in 2:1 ratio. So medians are 3/2 * distance.
Median ma = 3/2 * m1, etc. Then use the formula relating medians to area:
Area = (4/3) * sqrt(s*(s-ma)*(s-mb)*(s-mc)) where s = (ma+mb+mc)/2.

## Approach

Convert centroid-distances to medians (multiply by 3/2), then use Heron-like
formula for medians.
