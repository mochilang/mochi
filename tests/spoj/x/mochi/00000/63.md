# SPOJ Square Brackets (SQRBR)

https://www.spoj.com/problems/SQRBR/

Count sequences of N pairs of square brackets that are correctly matched,
given that some positions must contain '['.

## Approach

DP: dp[pos][open] = number of ways to complete from position pos with `open`
unmatched '[' so far. Forced '[' at certain positions constrains choices.
Process left to right; at each position place '[' (if open < remaining) or ']' (if open > 0).
Use big integers via string arithmetic or just store as int (fits in 64-bit for small N).
