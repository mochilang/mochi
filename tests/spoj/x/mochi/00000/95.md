# SPOJ Street Parade (STPAR)

https://www.spoj.com/problems/STPAR/

Cars arrive in order 1..N at a side street (stack). Given desired exit order,
determine if it's achievable using only the stack.

## Approach

Simulate: push each arriving car onto the stack, then pop while the stack top
equals the next required car. At the end check all cars exited.
