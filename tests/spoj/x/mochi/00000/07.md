# SPOJ BULK - The Bulk!

https://www.spoj.com/problems/BULK/

Given a 3D object defined by vertical (yz-plane-parallel) polygonal faces,
compute the volume of the solid using a coordinate-compression / ray casting approach.

## Approach

1. Parse all faces. Only vertical (constant x) faces are relevant for x-ray casting.
2. Coordinate compress x, y, z coordinates.
3. For each yz cell, cast an x-ray and count face crossings to determine inside/outside.
4. Sum volumes of cells that are inside.

## Notes

Complex geometry problem. Uses point-in-polygon for the yz cross-sections.
