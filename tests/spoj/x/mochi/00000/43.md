# SPOJ Copying Books (BOOKS1)

https://www.spoj.com/problems/BOOKS1/

Divide n chapters (contiguous) among k scribes to minimize the maximum number of pages
any scribe copies. Output the division (mark where each scribe starts).

## Approach

Binary search on the answer (max pages limit M):
- Check if we can split the chapters into at most k groups each with sum <= M.
- The smallest feasible M is the answer.
- Lower bound: max single chapter. Upper bound: sum of all chapters.

After finding M, greedily assign chapters left to right.
Output spaces between chapter boundaries, with '<' marking each division point.
