# SPOJ SBANK - Sorting Bank Accounts

https://www.spoj.com/problems/SBANK/

Read T test cases. Each has n bank account numbers (strings). Sort them
and print the sorted list with a blank line after each test case.

## Approach

Read all account strings, sort using insertion sort (or any stable sort),
print sorted. Each test case ends with a blank line in output.

## Notes

Account numbers are strings. Blank line separates input test cases too.
