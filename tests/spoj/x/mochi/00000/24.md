# SPOJ FCTRL2 - Small factorials

https://www.spoj.com/problems/FCTRL2/

Compute n! for n up to 100 and print all digits.

## Approach

Big integer multiplication represented as a digit list (least significant first).
Multiply by each number from 2 to n.

## Edge cases

- n=0 or n=1: answer is 1.
- Initialize digit list fresh each call (avoid shared literal bug in Mochi).
