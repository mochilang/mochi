# SPOJ PALIN - The Next Palindrome

https://www.spoj.com/problems/PALIN/

Given a number K (up to 10^6 digits), find the smallest palindrome strictly greater than K.

## Approach

1. If all digits are 9: answer is 10...01 (n+1 digits).
2. Mirror the left half onto the right half.
3. If the mirrored number is <= K, increment the middle digit(s) with carry.

## Notes

Big number handled as a digit array. O(n) per case.
