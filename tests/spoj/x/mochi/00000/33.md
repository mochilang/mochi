# SPOJ Trip (TRIP)

https://www.spoj.com/problems/TRIP/

Given two strings, find all distinct longest common subsequences (LCS) in
lexicographic order.

## Approach

1. Compute lcs[i][j] = LCS length of s[i:] and t[j:] via standard DP.
2. To enumerate all distinct LCS strings: for each character c in 'a'..'z',
   pick the leftmost occurrence in both s[si:] and t[tj:], recurse if it
   extends a valid LCS. Produces all distinct LCS strings without duplicates.
