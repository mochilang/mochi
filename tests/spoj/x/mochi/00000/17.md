# SPOJ Bytelandian Cryptographer (Act I) (CRYPTO1)

https://www.spoj.com/problems/CRYPTO1/

Given n, find two different positive integers a and b (a != b, a > 1, b > 1) such that
a * b = n. Print them in ascending order. If impossible, print "NO".

## Approach

Trial division from 2 to sqrt(n). The first factor found gives the pair (d, n/d).
Edge cases: n = 1 has no solution; prime n has no solution.
