# SPOJ Skyscraper (SCRAPER)

https://www.spoj.com/problems/SCRAPER/

A building has N floors (1..N). Starting at floor F, an elevator can move up A floors
or down B floors per move (staying within 1..N). Find the minimum number of moves to
reach floor T, or print "IMPOSSIBLE" if unreachable.

## Approach

BFS on floor numbers. Each floor is a node; edges go to floor+A (if <= N) and
floor-B (if >= 1). BFS gives the shortest path. Input terminates with all zeros.

## Sample

- `10 1 7 3 2`: floors 1..10, start=1, target=7, up=3, down=2 -> 1->4->7 = 2 moves
- `10 1 9 2 1`: 1->3->5->7->9 = 4 moves  
- `10 3 3 5 5`: already there = 0 moves
- `10 1 4 2 3`: IMPOSSIBLE (can only reach 1,3,5,7,9 going up then down only by 3)
