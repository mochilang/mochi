# SPOJ Dyzio (DYZIO)

https://www.spoj.com/problems/DYZIO/

Find the diameter of a weighted tree (the longest shortest path between any two nodes).

## Approach

Two BFS passes:
1. BFS from any node (node 0) to find the farthest node u.
2. BFS from u to find the farthest node v; the distance from u to v is the diameter.

Input: single integer n, then n-1 lines each with "u v w" (1-indexed edge with weight).
Multiple test cases until EOF.
