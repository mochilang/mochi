# SPOJ Bytelandian Cryptographer (Act II) (CRYPTO2)

https://www.spoj.com/problems/CRYPTO2/

Given an encoded sequence e[1..n] where e[i] = d[i-1] + d[i] + d[i+1] (boundaries 0),
find the original digits d[1..n] (each in 0..9), or print "Impossible".

## Approach

Try each possible value of d[1] from 0..9. Given d[1], d[2] = e[1] - d[1], and each
subsequent digit d[i+1] = e[i] - d[i-1] - d[i]. Check that all digits stay in 0..9
and that the virtual d[n+1] = 0 (boundary condition). Print the first valid solution
or "Impossible" if none found.

Input: T test cases. Each test case is a line of space-separated encoded values.
Output: space-separated original digits or "Impossible".
