# SPOJ BRCKTS - Brackets

https://www.spoj.com/problems/BRCKTS/

Given a bracket sequence, process queries: flip a bracket or check if the sequence is valid. Up to 10 test cases.

## Approach

Iterative segment tree storing (unmatched_open, unmatched_close) per node.
Merge(left, right): matched = min(left.open, right.close); result.open = left.open - matched + right.open; result.close = left.close + right.close - matched.
Valid iff root has (0, 0).

Query 0 = check validity, query i = flip bracket at position i (1-indexed).
