# SPOJ PLAY - Play on Words

https://www.spoj.com/problems/PLAY/

Given words, check if they can be chained so each word's first letter equals the last letter
of the previous word. Equivalent to: is the directed graph (letter -> letter) Eulerian?

## Approach

1. Build a directed graph where each word adds edge (first_char -> last_char).
2. Check connectivity of the underlying undirected graph using Union-Find.
3. Check Euler path/circuit conditions:
   - Euler circuit: all vertices with edges have in-degree == out-degree.
   - Euler path: exactly one vertex has out-degree - in-degree = 1 (start)
     and one has in-degree - out-degree = 1 (end), all others equal.
   - If neither: impossible.
