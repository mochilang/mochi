# SPOJ BEADS - Glass Beads

https://www.spoj.com/problems/BEADS/

Find the lexicographically smallest rotation of a string.

## Approach

Booth's algorithm (O(n)) or simple: concatenate string with itself,
find the minimum substring of length n starting at position 0..n-1.
Use the simple O(n^2) scan since n <= 10000 is manageable.

## Notes

Compare each rotation by comparing the doubled string.
