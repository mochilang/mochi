# SPOJ The Game of Master-Mind (MMIND)

https://www.spoj.com/problems/MMIND/

Given secret code and guess (both 4-digit sequences), compute:
- bulls: digits matching in exact position
- cows: digits present in code but in wrong position

## Approach

Count bulls directly. For cows: for each digit 0-9, accumulate min(count_in_code, count_in_guess),
then subtract bulls to get pure wrong-position matches.
