# SPOJ Numeral System of the Maya (MAYA)

https://www.spoj.com/problems/MAYA/

Convert a decimal number to Maya (base-20) representation using dots and bars.

## Approach

Decompose the number into base-20 digits. Each digit 0-19 is displayed as dots
(1 per unit mod 5, printed as ".") above bars (one "----" per group of 5).
Zero is shown as "----". Digits are printed most-significant first, separated
by a "----" divider line.
