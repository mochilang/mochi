# SPOJ ADDREV - Adding Reversed Numbers

https://www.spoj.com/problems/ADDREV/

Reverse each of two numbers, add them, then reverse the result (stripping leading zeros).

## Approach

1. Strip trailing zeros from the input strings (they become leading zeros after reversal).
2. Reverse both strings to get the numbers.
3. Add them using big integer string addition.
4. Reverse the result, strip leading zeros.

## Edge cases

- Numbers may have trailing zeros (leading zeros after reverse).
- Result may have trailing zeros after addition that become leading zeros after final reverse.
