# SPOJ Secret Code (CODE1)

https://www.spoj.com/problems/CODE1/

Convert a decimal integer N to base -2 (negabinary representation).

## Approach

Repeatedly divide by -2, collecting remainders (always 0 or 1). If the remainder
from the standard modulo operation is negative, add 2 to fix it. Collect digits
in reverse order. Special case: 0 maps to "0".

Example: 3 -> 111 (1*1 + 1*(-2) + 1*4 = 3), -3 -> 1101.
