# SPOJ Bus (BUS)

https://www.spoj.com/problems/BUS/

Given n circular bus routes, each a sequence of stop numbers repeated cyclically,
find the minimum time T such that all pairs of routes can mutually communicate
(i.e., from any route you can reach any other route by transferring at shared stops
at coinciding times). Output the maximum earliest arrival time across all (src, dst)
pairs, or "NEVER" if two routes never share a stop at the same time.

## Approach

For each pair of routes (i,j), find the first time they meet at the same stop
(period = LCM of their lengths). Then model as an all-pairs shortest path problem
where the "distance" from route i to route j starting at time t is the earliest
meeting time >= t. Use Dijkstra from each source route. Output the maximum of all
pairwise earliest-reachable times.
