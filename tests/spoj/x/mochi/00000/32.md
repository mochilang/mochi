# SPOJ NHAY - A Needle in the Haystack

https://www.spoj.com/problems/NHAY/

Find all occurrences of needle in haystack (0-indexed positions).

## Approach

KMP (Knuth-Morris-Pratt) for O(n+m) matching.
1. Build failure function for needle.
2. Scan haystack, output each match start position.

## Edge cases

- Multiple test cases separated by blank line.
- Output a blank line after each test case.
