# SPOJ Hang or not to hang (HANGLET)

https://www.spoj.com/problems/HANGLET/

Given a simple register machine with bitwise operations (AND, OR, XOR, NOT, MOV, SET,
RANDOM, JMP, JZ, STOP), determine whether it is guaranteed to terminate (STOP) starting
from unknown initial register values. If it always terminates, output the minimum number
of steps; otherwise output "HANGS".

## Approach

Use BFS over (pc, register state) pairs where each register is 0, 1, or "unknown" (2).
Start from pc=0 with all registers unknown (value 2). For each state, enumerate all
possible register value combinations (at most 2^k branches for RANDOM/unknown ops).
Track visited states to detect cycles. If STOP is reached, record the step count;
if all paths are explored without finding STOP, output "HANGS".
