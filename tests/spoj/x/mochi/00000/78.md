# SPOJ Marbles (MARBLES)

https://www.spoj.com/problems/MARBLES/

Given n identical marbles distributed into k distinct pots, each pot with at least
one marble, find the number of distributions.

## Approach

Stars and bars with minimum 1: C(n-1, k-1).
Compute via iterative multiplication/division: product((n-j)/(j) for j=1..k-1).
