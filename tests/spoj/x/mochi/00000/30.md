# SPOJ BLINNET - Bytelandian Blingors Network

https://www.spoj.com/problems/BLINNET/

Find the minimum cost to connect all cities (Minimum Spanning Tree).
Input: number of cities, then for each city: name, number of neighbors, (neighbor, cost) pairs.

## Approach

Kruskal's algorithm:
1. Parse all edges.
2. Sort edges by cost.
3. Use Union-Find to add edges greedily, skipping edges that would form cycles.
4. Output the total MST cost.
