# SPOJ PIR - Pyramids

https://www.spoj.com/problems/PIR/

Given n square pyramids each with base side a and height h, compute the volume
of each pyramid printed with 4 decimal places.

## Approach

Volume of a square pyramid = (1/3) * a^2 * h.

Input: n on first line, then n lines each with two integers a and h.
Output: volume for each pyramid, formatted to 4 decimal places.
