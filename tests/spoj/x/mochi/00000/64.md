# SPOJ Permutations (PERMUT1)

https://www.spoj.com/problems/PERMUT1/

Count alternating (zigzag) permutations of 1..N: p[1] < p[2] > p[3] < p[4] > ...

## Approach

DP: T[n][k] = number of alternating permutations of length n whose last element
has rank k. For even n, last step is down: T[n][k] = sum_{j>=k} T[n-1][j].
For odd n>=3, last step is up: T[n][k] = sum_{j<k} T[n-1][j].
E[n] = sum of T[n][k] over all k.
