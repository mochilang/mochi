# SPOJ PIGBANK - Piggy-Bank

https://www.spoj.com/problems/PIGBANK/

Piggy bank weighs E grams empty, F grams full. Given N coin types each with
weight w_i and value v_i, find the minimum value that exactly fills the
remaining weight (F - E), or output "This is impossible." if no exact solution.

## Approach

Unbounded knapsack DP. For each weight capacity 1..W, track the minimum value
to achieve exactly that weight. Use infinity for unreachable states.
