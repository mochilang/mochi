# SPOJ The Gordian Dance (DANCE)

https://www.spoj.com/problems/DANCE/

Given n cards, repeatedly perform a perfect in-shuffle: split the deck into two
equal halves, then interleave them starting with the bottom half (bottom card goes
to position 0, top card to position 1, etc.). Find how many shuffles are needed
to return to the original order.

## Approach

Compute the permutation induced by the in-shuffle:
- Card at position i (i < n/2) goes to position 2*i + 1
- Card at position i (i >= n/2) goes to position 2*(i - n/2)

Find all cycle lengths in this permutation via DFS, then return the LCM of all cycle
lengths. Uses Euclidean GCD. For odd n, the last card stays in place.
