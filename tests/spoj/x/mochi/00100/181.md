# SPOJ Scuba Diver (SCUBADIV)

https://www.spoj.com/problems/SCUBADIV/

Given target oxygen O and nitrogen N, find minimum weight set of cylinders
that provides at least O liters of oxygen and N liters of nitrogen.

## Approach

2D knapsack: dp[o][n] = min weight to get at least o liters of O2 and n liters of N2.
Cap coordinates at target values to keep table size O*N.
