# SPOJ Tower of Babylon (BABTWR)

https://www.spoj.com/problems/BABTWR/

Given n block types, each with three dimensions, stack blocks to maximize height.
A block can sit on top of another if its base (any two dimensions) strictly fits
within the other's base.

## Approach

For each block type, generate all 3 orientations (choose which dim is height).
Sort by base area descending. DP: dp[i] = max tower height with block i on top.
