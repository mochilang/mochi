# SPOJ Rent Your Airplane (RENT)

https://www.spoj.com/problems/RENT/

Given N rental intervals (start, end, profit), choose non-overlapping intervals
to maximize total profit.

## Approach

Weighted interval scheduling: sort by end time. DP: dp[i] = max profit using
first i intervals. dp[i] = max(dp[i-1], profit[i] + dp[last_compatible(i)]).
Binary search for last_compatible.
