# SPOJ The Last Digit (LASTDIG)

https://www.spoj.com/problems/LASTDIG/

Compute the last digit of A^B.

## Approach

Last digit of powers cycle with period 1 or 4 (depends on A%10).
Special cases: B=0 → 1, A ends in 0/1/5/6 → constant last digit.
For others: use (B-1)%4 + 1 to find position in the 4-cycle.
