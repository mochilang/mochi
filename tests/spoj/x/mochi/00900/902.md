# SPOJ Hangover (HANGOVER)

https://www.spoj.com/problems/HANGOVER/

Find the minimum number of cards in a stack so that the top card overhangs
the table by at least d. The k-th card from top contributes 1/(2k) to overhang.

## Approach

Accumulate sum = 1/2 + 1/4 + 1/6 + ... until sum >= d. Count cards.
