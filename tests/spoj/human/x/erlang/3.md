# [Substring Check (Bug Funny)](https://www.spoj.com/problems/SBSTR1)

## Algorithm
1. Read pairs of binary strings `A` (length 10) and `B` (length 5).
2. For each pair, search for `B` within `A` using substring search.
3. Output `1` if found, otherwise `0`.

This runs in O(|A|) time per pair with constant extra space.
