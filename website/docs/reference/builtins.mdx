These functions are available in every Mochi program without any imports.

## Output

### `print(values...)`

Print values to stdout, separated by spaces, followed by a newline:

```mochi
print("Hello", "world")   // Hello world
print(1 + 2)              // 3
```

### `eprint(values...)`

Print to stderr:

```mochi
eprint("Warning:", message)
```

## Type conversions

### `str(value): string`

Convert any value to its string representation:

```mochi
str(42)        // "42"
str(3.14)      // "3.14"
str(true)      // "true"
str([1,2,3])   // "[1, 2, 3]"
```

### `int(value): int`

Parse a string to integer, or truncate a float:

```mochi
int("42")    // 42
int(3.99)    // 3
```

### `float(value): float`

Parse or convert to float:

```mochi
float("3.14")   // 3.14
float(42)       // 42.0
```

### `bool(value): bool`

Convert to boolean:

```mochi
bool(0)      // false
bool(1)      // true
bool("")     // false
bool("hi")   // true
```

## Collections

### `len(collection): int`

Return the length of a list, string, map, or set:

```mochi
len([1, 2, 3])      // 3
len("hello")        // 5
len({"a": 1})       // 1
```

### `keys(map): [K]`

Return the keys of a map as a list:

```mochi
keys({"a": 1, "b": 2})   // ["a", "b"]
```

### `values(map): [V]`

Return the values of a map:

```mochi
values({"a": 1, "b": 2})   // [1, 2]
```

### `push(list, value)`

Append a value to a list (mutates in place):

```mochi
var nums = [1, 2]
push(nums, 3)
print(nums)   // [1, 2, 3]
```

### `pop(list): T`

Remove and return the last element:

```mochi
var nums = [1, 2, 3]
let last = pop(nums)
print(last)   // 3
print(nums)   // [1, 2]
```

### `contains(collection, value): bool`

Check membership (equivalent to `value in collection`):

```mochi
contains([1,2,3], 2)   // true
```

### `slice(list, start, end): [T]`

Return a sub-list:

```mochi
slice([1,2,3,4,5], 1, 4)   // [2, 3, 4]
```

## Functional

### `map(list, fn): [U]`

Apply a function to each element:

```mochi
map([1,2,3], fun(n: int): int => n * 2)   // [2, 4, 6]
```

### `filter(list, fn): [T]`

Keep elements for which `fn` returns `true`:

```mochi
filter([1,2,3,4,5], fun(n: int): bool => n % 2 == 0)   // [2, 4]
```

### `reduce(list, fn, initial): U`

Fold a list into a single value:

```mochi
reduce([1,2,3,4,5], fun(acc: int, n: int): int => acc + n, 0)   // 15
```

### `sort(list): [T]`

Return a sorted copy of the list:

```mochi
sort([3,1,4,1,5])   // [1, 1, 3, 4, 5]
```

### `sort_by(list, fn): [T]`

Sort by a key function:

```mochi
sort_by(users, fun(u: User): int => u.age)
```

### `reverse(list): [T]`

Return a reversed copy:

```mochi
reverse([1,2,3])   // [3, 2, 1]
```

## Math

### `abs(n): T`

Absolute value:

```mochi
abs(-5)    // 5
abs(-3.0)  // 3.0
```

### `min(a, b): T` / `max(a, b): T`

Minimum / maximum of two values:

```mochi
min(3, 7)   // 3
max(3, 7)   // 7
```

### `floor(n): int` / `ceil(n): int`

Round toward negative / positive infinity:

```mochi
floor(3.7)   // 3
ceil(3.2)    // 4
```

### `round(n): int`

Round to nearest integer:

```mochi
round(3.5)   // 4
```

### `sqrt(n): float`

Square root:

```mochi
sqrt(16.0)   // 4.0
```

## Strings

### `split(s, sep): [string]`

Split a string by separator:

```mochi
split("a,b,c", ",")   // ["a", "b", "c"]
```

### `join(list, sep): string`

Join a list of strings:

```mochi
join(["a", "b", "c"], "-")   // "a-b-c"
```

### `trim(s): string`

Remove leading/trailing whitespace:

```mochi
trim("  hello  ")   // "hello"
```

### `upper(s)` / `lower(s)`

Change case:

```mochi
upper("hello")   // "HELLO"
lower("HELLO")   // "hello"
```

### `starts_with(s, prefix): bool` / `ends_with(s, suffix): bool`

```mochi
starts_with("hello", "he")   // true
ends_with("world", "ld")     // true
```

### `replace(s, old, new): string`

```mochi
replace("foo bar foo", "foo", "baz")   // "baz bar baz"
```

## I/O

### `read_file(path): string | nil`

Read a file as a string. Returns `nil` if the file doesn't exist.

### `write_file(path, content: string)`

Write a string to a file (creates or overwrites).

### `read_line(): string`

Read a line from stdin.

## Misc

### `type_of(value): string`

Return the type name as a string:

```mochi
type_of(42)        // "int"
type_of("hello")   // "string"
type_of([1,2])     // "list"
```

### `panic(message: string)`

Print message and terminate the program:

```mochi
panic("unreachable code reached")
```

### `assert(condition: bool, message: string)`

Panic with message if condition is false:

```mochi
assert(x > 0, "x must be positive")
```
