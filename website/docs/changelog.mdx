## Recent releases

### v0.10.x (current)

- Expanded standard library and built-in functions.
- Added hundreds of algorithm implementations under `examples/`.
- Improved bytecode optimizer.
- Stabilized Docker image (`ghcr.io/mochilang/mochi`).
- Improved MCP server.

### v0.9.x

- Improved type inference for union types.
- Added `from` / `where` / `select` / `join` dataset queries.
- Added `load` and `save` for CSV, JSON, JSONL, and YAML.
- Added the pipe operator `|` for functional composition.

### v0.8.x

- Stabilized agents and streams.
- Added `intent` blocks for MCP tool exposure.
- Added the `mochi serve` command.

### v0.7.x

- Added `generate` blocks for AI text generation.
- Added `generate embedding` for vector embeddings.
- Added `model` configuration blocks.
- Added tool calling inside `generate` blocks.

### v0.6.x

- Added the package system (`package`, `export`, `import`).
- Added FFI through `extern` declarations.
- Added transpilation to Go, Python, and TypeScript.

### v0.5.x

- Added agents (`agent` blocks with `on` handlers).
- Added streams (`stream` definitions, `emit`).
- Added MCP integration in `mochi serve`.

### v0.4.x

- Added union types (`T | nil`, `A | B`).
- Added `match` expressions.
- Added destructuring assignment.

### v0.3.x

- Added type declarations (`type`).
- Added closures and arrow functions.
- Added `for` and `while` loops with `break` and `continue`.

### v0.2.x

- Added static type checking.
- Added the `let` / `var` distinction.
- Added built-in test blocks (`test` / `expect`).

### v0.1.x

- Initial release.
- Added bytecode VM.
- Added `fun`, `if` / `else`, lists, and maps.
- Added `print`, `str`, and `len` built-ins.
