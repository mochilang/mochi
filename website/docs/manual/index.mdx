Welcome to the Mochi Manual — your complete guide to the Mochi programming language.

Mochi is a small, statically typed language built for clarity, safety, and expressiveness.
It blends clean declarative syntax with first-class support for agents, streams, datasets, and AI generation.

If you see anything that can be improved, please
[open an issue](https://github.com/mochilang/mochi/issues/new) or send a pull request.
