# TypeScript AST Printer

This directory contains utilities for inspecting TypeScript source with Tree-sitter
and printing it back. Golden tests ensure round-tripping for a subset of samples.

## Golden samples

- [x] append_builtin.ts
- [x] avg_builtin.ts
- [x] basic_compare.ts
