# Jul 2025 (v0.10.16)

Released on Sat Jul 5 16:16:38 2025 +0700.

Mochi v0.10.16 introduces cross-language conversion tools and sandboxed execution while expanding SQL Logic Test coverage.

## Runtime
- `min` and `max` builtins added to the C backend
- Aggregator functions extended for SQL Logic Test datasets

## Compilers
- Generic `any2mochi` translator with JSON diagnostics and server mode
- Python, Go and TypeScript converters updated with broader language support
- Fortran `upper` and `lower` builtins
- Scala map field access and Clojure compiler fixes

## Datasets
- SQL Logic Test generator handles SELECT ALL and group-by cases 300â€“499
- Additional conversion cases stored for language-translated Mochi files

## Tooling
- sqlite2duck utility gains date, string and boolean conversions with CLI tests
- Runner protocol with multi-stage sandbox containers
